# API Test Examples: Order Creation to Payment Flow

This file contains example API requests to test the complete flow from adding items to cart, creating an order, and processing payment.

## Prerequisites
- Run `php artisan db:seed` to populate the database with test data
- User must be authenticated with Sanctum token
- Replace `YOUR_TOKEN` with actual bearer token
- Replace `BASE_URL` with your API base URL (e.g., http://localhost:8000/api)

## Authentication

### Register a new user (optional)
```bash
curl -X POST BASE_URL/register \
  -H "Content-Type: application/json" \
  -d '{
    "name": "New User",
    "email": "newuser@example.com",
    "password": "password123",
    "password_confirmation": "password123"
  }'
```

### Login to get token
```bash
curl -X POST BASE_URL/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "test@example.com",
    "password": "password"
  }'
```

Expected response:
```json
{
  "user": {
    "id": 1,
    "name": "Test User",
    "email": "test@example.com",
    "email_verified_at": null,
    "created_at": "2024-01-07T10:00:00.000000Z",
    "updated_at": "2024-01-07T10:00:00.000000Z"
  },
  "token": "YOUR_TOKEN_HERE"
}
```

### Logout (optional)
```bash
curl -X POST BASE_URL/logout \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## 1. Add Items to Cart

### Add Product to Cart (Wireless Headphones)
```bash
curl -X POST BASE_URL/cart/add \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "product_id": 1,
    "quantity": 2
  }'
```

### Add Another Product to Cart (Cotton T-Shirt)
```bash
curl -X POST BASE_URL/cart/add \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "product_id": 3,
    "quantity": 1
  }'
```

### Get Cart Summary
```bash
curl -X GET BASE_URL/cart/summary \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## 2. Create Order

### Create Order from Cart
```bash
curl -X POST BASE_URL/orders \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "shipping_address": {
      "street": "123 Main St",
      "city": "Cairo",
      "state": "Cairo",
      "postal_code": "12345",
      "country": "Egypt"
    },
    "notes": "Please handle with care"
  }'
```

### Alternative: Create Order with Specific Items (bypasses cart)
```bash
curl -X POST BASE_URL/orders \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "items": [
      {
        "product_id": 1,
        "quantity": 2
      },
      {
        "product_id": 3,
        "quantity": 1
      }
    ],
    "shipping_address": {
      "street": "123 Main St",
      "city": "Cairo",
      "state": "Cairo",
      "postal_code": "12345",
      "country": "Egypt"
    },
    "notes": "Please handle with care"
  }'
```

### Get Order Details
```bash
curl -X GET BASE_URL/orders/1 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## 3. Process Payment

### Process Payment for Order
```bash
curl -X POST BASE_URL/payment/process \
  -H "Content-Type: application/json" \
  -d '{
    "order_id": 1,
    "payment_method": "card",
    "card_details": {
      "number": "4111111111111111",
      "expiry_month": "12",
      "expiry_year": "2025",
      "cvv": "123"
    },
    "billing_address": {
      "street": "123 Main St",
      "city": "Cairo",
      "state": "Cairo",
      "postal_code": "12345",
      "country": "Egypt"
    }
  }'
```

### Payment Callback (handled by payment gateway)
```bash
curl -X POST BASE_URL/payment/callback \
  -H "Content-Type: application/json" \
  -d '{
    "transaction_id": "txn_123456",
    "order_id": 1,
    "status": "success",
    "amount": 328.96
  }'
```

## 4. Optional: Shipment Creation (after payment success)

### Create Shipment
```bash
curl -X POST BASE_URL/shipments \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "order_id": 1,
    "receiver": {
      "firstName": "John",
      "lastName": "Doe",
      "phone": "+201234567890",
      "email": "john@example.com"
    },
    "dropOffAddress": {
      "city": "Cairo",
      "zoneId": "1",
      "districtId": "1",
      "firstLine": "123 Main St",
      "secondLine": "Apartment 4B",
      "buildingNumber": "123",
      "floor": "4",
      "apartment": "4B"
    },
    "cod": 328.96
  }'
```

### Track Shipment
```bash
curl -X GET BASE_URL/shipments/TRACKING_NUMBER
```
